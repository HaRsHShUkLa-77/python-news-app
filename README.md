# 📰 News App (Python + Requests)

A simple command-line News Application built with Python using the Requests module and the NewsAPI.

## 🚀 Features

- View Top Headlines
- General News
- Sports News
- Technology News
- Business News
- Entertainment News
- Top 10 News Display
- Exception Handling
- HTTP Error Handling
- Menu Driven Interface

## 🛠 Technologies Used

- Python 3
- Requests Library
- NewsAPI

## 📂 Project Structure

```
News-App/
│
├── main.py
├── README.md
└── requirements.txt
```

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/python-news-app.git
```

Go to the project folder

```bash
cd python-news-app
```

Install Requests

```bash
pip install requests
```

## ▶️ Run the Program

```bash
python main.py
```

Enter your NewsAPI key when prompted.

Example:

```
Enter your API key From NEWS API:
```
